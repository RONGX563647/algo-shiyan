## 二、实验内容

### （一）相等元素问题

**问题描述**：给出一个整数集合，假定这些整数存储在数组A[1…n]中，确定它们中是否存在两个相等的元素。

**输入格式**：
- 第一行：正整数m，表示测试例个数
- 每个测试例包含两行：
  - 第一行：正整数n (n≤500)，表示整数序列的长度
  - 第二行：整数序列，整数之间用一个空格隔开

**输出格式**：对于每个测试例输出一行，若存在两个相等的元素则输出Yes，否则输出No。

### （二）整数集合分解

**问题描述**：将n个正整数的集合分成两个子集S1和S2，每个子集含有n/2个元素，使S1中所有元素的和与S2中所有元素的和的差最大。

**输入格式**：
- 第一行：正整数m，表示测试例个数
- 每个测试例包含两行：
  - 第一行：正整数n (n≤500)，表示原整数集合的长度
  - 第二行：n个整数序列，整数之间用一个空格隔开

**输出格式**：
- 第一行：元素和较小的整数集合（从小到大排序）
- 第二行：元素和较大的整数集合（从小到大排序）
- 两个测试例之间用一个空行隔开

## 四、实验步骤

### （一）相等元素问题

#### 1. 算法设计分析

**方法比较**：

| 方法 | 思路 | 时间复杂度 | 空间复杂度 |
|------|------|------------|------------|
| 暴力法 | 两层循环比较每对元素 | O(n²) | O(1) |
| 排序法 | 排序后比较相邻元素 | O(n log n) | O(1)或O(n) |
| 哈希集合 | 使用HashSet存储已遍历元素 | O(n)平均 | O(n) |

**选择方法**：采用哈希集合方法，效率最高。本实验**自实现HashSet**，不使用Java标准库容器。

#### 2. 自实现HashSet设计

**数据结构**：采用链地址法（Separate Chaining）解决哈希冲突

```
哈希表结构：
┌─────┐    ┌─────┐    ┌─────┐
│  0  │───→│  5  │───→│ 21  │───→ null
├─────┤    └─────┘    └─────┘
│  1  │───→ null
├─────┤    ┌─────┐    ┌─────┐
│  2  │───→│ 16  │───→│  2  │───→ null
├─────┤    └─────┘    └─────┘
│ ... │
└─────┘
```

**核心代码实现**：

```java
// 链表节点定义
private static class Node {
    int value;
    Node next;
    Node(int value) {
        this.value = value;
        this.next = null;
    }
}

// 哈希函数
private int hash(int value) {
    return Math.abs(value) % capacity;
}

// 添加元素（返回false表示已存在）
public boolean add(int value) {
    int index = hash(value);

    // 遍历链表检查是否已存在
    Node current = table[index];
    while (current != null) {
        if (current.value == value) {
            return false;  // 元素已存在
        }
        current = current.next;
    }

    // 添加新节点到链表头部
    Node newNode = new Node(value);
    newNode.next = table[index];
    table[index] = newNode;
    return true;
}
```

**检测重复元素**：

```java
public static boolean hasDuplicate(int[] arr) {
    // 创建哈希表，容量设为数组长度的2倍以减少冲突
    MyHashSet set = new MyHashSet(arr.length * 2);

    for (int num : arr) {
        if (!set.add(num)) {
            return true;  // 发现重复元素
        }
    }
    return false;
}
```

- **时间复杂度**：

  - 最坏情况：O(n²)，当所有元素哈希冲突到同一个桶时

- **空间复杂度**：O(n)，需要存储n个元素的哈希表

### （二）整数集合分解

#### 1. 算法设计分析

**贪心策略**：要使两个子集的和差最大，应该让一个子集包含最小的n/2个元素，另一个包含最大的n/2个元素。

**正确性证明**：

设排序后数组为 a[0] ≤ a[1] ≤ ... ≤ a[n-1]

将数组分成两部分：
- S1 = {a[0], a[1], ..., a[n/2-1]}（最小的一半元素）
- S2 = {a[n/2], a[n/2+1], ..., a[n-1]}（最大的一半元素）

对于任意其他分割方式，如果将S2中的某个元素x与S1中的某个元素y交换（其中y < x），则：
- 新S1和 = 原S1和 - y + x
- 新S2和 = 原S2和 - x + y
- 新差值 = |原差值 - 2(x-y)| < 原差值

因此，将最小的一半和最大的一半分组的分割方式确实能获得最大差值。

#### 2. 自实现快速排序算法

**快速排序原理**：
1. 选择一个基准元素（pivot）
2. 将小于基准的元素移到左边，大于基准的元素移到右边
3. 递归处理左右两个子数组

**算法图示**：

```
原始数组: [68, 25, 34, 16, 2, 37, 3, 95]

第一轮分区 (pivot=95):
[68, 25, 34, 16, 2, 37, 3, 95]
                          ↑基准位置

第二轮分区 (左半部分, pivot=3):
[2, 3, 34, 16, 25, 37, 68, 95]
    ↑基准位置

继续递归...

最终结果: [2, 3, 16, 25, 34, 37, 68, 95]
```

**核心代码实现**：

```java
// 快速排序主方法
public static void quickSort(int[] arr) {
    if (arr == null || arr.length <= 1) {
        return;
    }
    quickSort(arr, 0, arr.length - 1);
}

// 快速排序递归方法
private static void quickSort(int[] arr, int low, int high) {
    if (low < high) {
        int pivotIndex = partition(arr, low, high);
        quickSort(arr, low, pivotIndex - 1);
        quickSort(arr, pivotIndex + 1, high);
    }
}

// 分区函数
private static int partition(int[] arr, int low, int high) {
    int pivot = arr[high];  // 选择最右元素作为基准
    int i = low - 1;        // i指向小于基准的最后一个元素

    for (int j = low; j < high; j++) {
        if (arr[j] <= pivot) {
            i++;
            swap(arr, i, j);
        }
    }
    swap(arr, i + 1, high);
    return i + 1;
}

// 交换数组中两个元素
private static void swap(int[] arr, int i, int j) {
    if (i != j) {
        int temp = arr[i];
        arr[i] = arr[j];
        arr[j] = temp;
    }
}
```

**分割数组**：

```java
public static int[][] partition(int[] arr) {
    int n = arr.length;

    // 复制数组避免修改原数组
    int[] sorted = copyArray(arr);

    // 使用自实现的快速排序
    quickSort(sorted);

    // 分割数组
    int half = n / 2;
    int[] smaller = new int[half];
    int[] larger = new int[half];

    for (int i = 0; i < half; i++) {
        smaller[i] = sorted[i];        // 前半部分（较小元素）
        larger[i] = sorted[half + i];  // 后半部分（较大元素）
    }

    return new int[][]{smaller, larger};
}
```

#### 3. 算法复杂度分析

**快速排序**：
- 时间复杂度：
  - 平均情况：O(n log n)
  - 最坏情况：O(n²)（当数组已排序或逆序时）
- 空间复杂度：O(log n)（递归调用栈）

**整体分割算法**：
- 时间复杂度：O(n log n)（主要是排序）
- 空间复杂度：O(n)（存储分割后的数组）

## 六、总结

### 实验过程中遇到的问题及解决办法

1. **哈希函数设计**：
   - 问题：负整数的哈希值可能为负
   - 解决：使用 `Math.abs(value) % capacity` 确保索引非负

2. **哈希冲突处理**：
   - 问题：不同元素可能映射到同一位置
   - 解决：采用链地址法，每个桶使用链表存储冲突元素

3. **快速排序基准选择**：
   - 问题：最坏情况下（已排序数组）时间复杂度退化为O(n²)
   - 解决：本题数据量较小（n≤500），性能影响不大；实际应用可采用随机化或三数取中法优化

### EqualElements.java（相等元素问题）

```java
package com.rongx;

import java.util.Scanner;

/**
 * 实验一：相等元素问题
 * 使用自实现的HashSet检测重复元素
 */
public class EqualElements {

    /**
     * 自定义HashSet实现
     * 使用链地址法解决哈希冲突
     */
    static class MyHashSet {
        // 链表节点
        private static class Node {
            int value;
            Node next;

            Node(int value) {
                this.value = value;
                this.next = null;
            }
        }

        private Node[] table;  // 哈希表数组
        private int capacity;  // 容量

        public MyHashSet(int capacity) {
            this.capacity = capacity;
            this.table = new Node[capacity];
        }

        // 哈希函数
        private int hash(int value) {
            return Math.abs(value) % capacity;
        }

        /**
         * 添加元素
         * @param value 要添加的值
         * @return 如果元素已存在返回false，否则添加并返回true
         */
        public boolean add(int value) {
            int index = hash(value);

            // 检查是否已存在
            Node current = table[index];
            while (current != null) {
                if (current.value == value) {
                    return false;  // 元素已存在
                }
                current = current.next;
            }

            // 添加新节点到链表头部
            Node newNode = new Node(value);
            newNode.next = table[index];
            table[index] = newNode;
            return true;
        }
    }

    /**
     * 检测数组中是否存在重复元素
     * @param arr 输入整数数组
     * @return 如果存在重复元素返回true，否则返回false
     */
    public static boolean hasDuplicate(int[] arr) {
        MyHashSet set = new MyHashSet(arr.length * 2);

        for (int num : arr) {
            if (!set.add(num)) {
                return true;
            }
        }
        return false;
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int m = scanner.nextInt();

        for (int i = 0; i < m; i++) {
            int n = scanner.nextInt();
            int[] arr = new int[n];
            for (int j = 0; j < n; j++) {
                arr[j] = scanner.nextInt();
            }

            if (hasDuplicate(arr)) {
                System.out.println("Yes");
            } else {
                System.out.println("No");
            }
        }
        scanner.close();
    }
}
```

### SetPartition.java（整数集合分解）

```java
package com.rongx;

import java.util.Scanner;

/**
 * 实验二：整数集合分解
 * 使用自实现的快速排序算法
 */
public class SetPartition {

    /**
     * 快速排序主方法
     */
    public static void quickSort(int[] arr) {
        if (arr == null || arr.length <= 1) {
            return;
        }
        quickSort(arr, 0, arr.length - 1);
    }

    /**
     * 快速排序递归方法
     */
    private static void quickSort(int[] arr, int low, int high) {
        if (low < high) {
            int pivotIndex = partition(arr, low, high);
            quickSort(arr, low, pivotIndex - 1);
            quickSort(arr, pivotIndex + 1, high);
        }
    }

    /**
     * 分区函数：选择最右元素作为基准
     */
    private static int partition(int[] arr, int low, int high) {
        int pivot = arr[high];
        int i = low - 1;

        for (int j = low; j < high; j++) {
            if (arr[j] <= pivot) {
                i++;
                swap(arr, i, j);
            }
        }
        swap(arr, i + 1, high);
        return i + 1;
    }

    /**
     * 交换数组中两个元素
     */
    private static void swap(int[] arr, int i, int j) {
        if (i != j) {
            int temp = arr[i];
            arr[i] = arr[j];
            arr[j] = temp;
        }
    }

    /**
     * 复制数组
     */
    public static int[] copyArray(int[] arr) {
        int[] copy = new int[arr.length];
        for (int i = 0; i < arr.length; i++) {
            copy[i] = arr[i];
        }
        return copy;
    }

    /**
     * 分割数组
     */
    public static int[][] partition(int[] arr) {
        int n = arr.length;
        int[] sorted = copyArray(arr);
        quickSort(sorted);

        int half = n / 2;
        int[] smaller = new int[half];
        int[] larger = new int[half];

        for (int i = 0; i < half; i++) {
            smaller[i] = sorted[i];
            larger[i] = sorted[half + i];
        }
        return new int[][]{smaller, larger};
    }

    /**
     * 打印数组
     */
    public static void printArray(int[] arr) {
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < arr.length; i++) {
            if (i > 0) sb.append(" ");
            sb.append(arr[i]);
        }
        System.out.println(sb.toString());
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int m = scanner.nextInt();

        for (int i = 0; i < m; i++) {
            int n = scanner.nextInt();
            int[] arr = new int[n];
            for (int j = 0; j < n; j++) {
                arr[j] = scanner.nextInt();
            }

            int[][] result = partition(arr);
            printArray(result[0]);
            printArray(result[1]);

            if (i < m - 1) {
                System.out.println();
            }
        }
        scanner.close();
    }
}
```
