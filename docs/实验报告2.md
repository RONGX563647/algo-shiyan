## 二、实验内容

### （一）格雷码问题

**问题描述**：对于给定的正整数n，格雷码为满足如下条件的一个编码序列：

1. 序列由2^n个编码组成，每个编码都是长度为n的二进制位串
2. 序列中无相同的编码
3. 序列中位置相邻的两个编码恰有一位不同

**输入格式**：

- 第一行：正整数m，表示测试例个数
- 每个测试例包含一个正整数n

**输出格式**：

- 对于每个测试例n，输出2^n个长度为n的格雷码
- 每个格雷码内，两个位之间用一个空格隔开
- 两个测试例之间用空行隔开，最后一个测试例后无空行

### （二）棋子移动游戏

**问题描述**：有2n（n≥4为整数）棋子，黑白棋子各n个。

**初始状态**：先白后黑，右边两空位

```
W W W ... W B B B ... B _ _
```

**终止状态**：白黑相间，左边两空位

```
_ _ W B W B W B ...
```

**移动规则**：

1. 每次必须同时移动相邻两个棋子
2. 移动方向不限
3. 每次移动必须跳过若干棋子

## 四、实验步骤

### （一）格雷码问题

#### 1. 算法设计分析

**递归关系推导**：

n位格雷码可以由n-1位格雷码构造：

1. 获取n-1位格雷码序列
2. 在每个n-1位格雷码前加"0"（前半部分）
3. 将n-1位格雷码逆序，每个元素前加"1"（后半部分）
4. 拼接两部分

**递归示意图**：

```
n=1:  0, 1

n=2:  前半部分(加0): 00, 01
      后半部分(逆序+1): 11, 10
      结果: 00, 01, 11, 10

n=3:  前半部分(加0): 000, 001, 011, 010
      后半部分(逆序+1): 110, 111, 101, 100
      结果: 000, 001, 011, 010, 110, 111, 101, 100
```

**正确性证明**：

设G(n)为n位格雷码序列，G(n-1)为n-1位格雷码序列。

1. **数量正确**：|G(n)| = 2|G(n-1)| = 2·2^(n-1) = 2^n
2. **无重复**：
   - 前半部分以"0"开头，后半部分以"1"开头，两部分无交集
   - 前半部分由G(n-1)元素前加"0"，G(n-1)无重复，故前半部分无重复
   - 后半部分同理
3. **相邻恰有一位不同**：
   - 前半部分内部：由G(n-1)加"0"，G(n-1)相邻恰有一位不同，加"0"后仍是
   - 后半部分内部：由G(n-1)逆序加"1"，相邻关系保持
   - 前后半部分交界处：前半部分最后元素是"0"+G(n-1)最后元素，后半部分第一元素是"1"+G(n-1)最后元素，恰有一位（首位）不同

#### 2. 核心代码实现

```java
/**
 * 递归生成n位格雷码
 * @param n 格雷码位数
 * @return 格雷码列表
 */
public static List<String> generateGrayCode(int n) {
    List<String> result = new ArrayList<>();

    // 递归出口：n=1时直接返回基础格雷码
    if (n == 1) {
        result.add("0");
        result.add("1");
        return result;
    }

    // 递归调用：获取n-1位格雷码
    List<String> prevGrayCode = generateGrayCode(n - 1);

    // 构造前半部分：在每个格雷码前加"0"
    for (String code : prevGrayCode) {
        result.add("0" + code);
    }

    // 构造后半部分：逆序遍历，在每个格雷码前加"1"
    for (int i = prevGrayCode.size() - 1; i >= 0; i--) {
        result.add("1" + prevGrayCode.get(i));
    }

    return result;
}
```

#### 3. 算法复杂度分析

- **时间复杂度**：O(n·2^n)
  - 递归深度为n
  - 第k层产生2^k个格雷码，每个格雷码长度为k
  - 总时间 = Σ(k=1 to n) k·2^k = O(n·2^n)
- **空间复杂度**：O(n·2^n)
  - 存储2^n个长度为n的格雷码
  - 递归调用栈深度为n

### （二）棋子移动游戏

#### 1. 算法设计分析

**问题建模**：

将棋子移动问题转化为**状态空间搜索**问题：

- **状态**：棋盘的当前布局，用字符串表示（如"WWWWBBBB\_\_"）
- **初始状态**：n个W + n个B + "\_\_"
- **目标状态**："\_\_" + n组"WB"
- **状态转移**：选择相邻两个棋子，跳过若干棋子移动到空位

**算法选择**：

| 方法        | 特点        | 时间复杂度               |
| --------- | --------- | ------------------- |
| DFS（深度优先） | 找到的解不一定最短 | O(b^d)，b为分支因子，d为解深度 |
| BFS（广度优先） | 保证找到最短路径  | O(b^d)，但保证最优        |

选择**BFS广度优先搜索**，保证找到最短解。

#### 2. BFS算法图解

```
初始状态: W W W W B B B B _ _
          ↓
      ┌───┴───┐
      ↓       ↓
W W W _ _ B B B W B   ... (其他可能移动)
      ↓
  继续扩展...
      ↓
目标状态: _ _ W B W B W B W B
```

**BFS搜索过程**：

```
队列状态变化:
初始: [WWWWBBBB__]
第1轮: [WWW__BBBW B, W_WWBBBBW B, ...]  (扩展初始状态)
第2轮: [...] (扩展第1轮状态)
...
第5轮: [__WBWBWBWB]  (找到目标状态!)
```

#### 3. 核心代码实现

```java
/**
 * 使用BFS求解棋子移动问题的最短路径
 */
public List<String> solve() {
    Queue<String> queue = new LinkedList<>();
    Map<String, String> parent = new HashMap<>();

    queue.offer(initialState);
    parent.put(initialState, null);

    while (!queue.isEmpty()) {
        String current = queue.poll();

        if (current.equals(targetState)) {
            return reconstructPath(parent, current);
        }

        // 获取所有可能的下一状态
        List<String> nextStates = getNextStates(current);
        for (String next : nextStates) {
            if (!parent.containsKey(next)) {
                parent.put(next, current);
                queue.offer(next);
            }
        }
    }
    return null;
}

/**
 * 获取当前状态所有可能的下一状态
 */
private List<String> getNextStates(String state) {
    List<String> nextStates = new ArrayList<>();
    int emptyPos = state.indexOf('_');

    for (int i = 0; i < state.length() - 1; i++) {
        // 检查位置i和i+1是否都是棋子
        if (state.charAt(i) != '_' && state.charAt(i + 1) != '_') {
            // 检查是否跳过了棋子
            int minPos = Math.min(i, emptyPos);
            int maxPos = Math.max(i, emptyPos);
            boolean hasJump = false;
            for (int j = minPos + 1; j < maxPos; j++) {
                if (state.charAt(j) != '_') {
                    hasJump = true;
                    break;
                }
            }

            if (hasJump) {
                String newState = move(state, i, emptyPos);
                nextStates.add(newState);
            }
        }
    }
    return nextStates;
}
```

#### 4. 算法复杂度分析

- **状态空间大小**：O(C(2n+2, 2)) = O(n²)
  - 总位置数为2n+2
  - 需要选择2个空位位置
- **时间复杂度**：O(n²·n) = O(n³)
  - 每个状态最多有O(n)种移动
  - 状态数为O(n²)
- **空间复杂度**：O(n²·n) = O(n³)
  - 存储O(n²)个状态，每个状态长度为O(n)

## 六、总结

### 实验过程中遇到的问题及解决办法

1. **格雷码递归设计**：
   - 问题：如何保证相邻编码恰有一位不同
   - 解决：通过逆序拼接的方式，保证前后半部分交界处只差一位
2. **棋子移动搜索效率**：
   - 问题：使用DFS找到的解很长（n=4时96步）
   - 解决：改用BFS广度优先搜索，保证找到最短路径（n=4时仅需5步）
3. **状态表示与去重**：
   - 问题：搜索过程中可能重复访问同一状态
   - 解决：使用HashMap记录已访问状态，避免重复搜索

### 个人收获

1. **递归思想深入理解**：
   - 掌握了递归三要素：递归关系、递归出口、参数设置
   - 学会了将复杂问题分解为相同性质的子问题
2. **状态空间搜索**：
   - 理解了DFS和BFS的特点与适用场景
   - BFS适合寻找最短路径，DFS适合寻找任意解或路径存在性
3. **算法复杂度分析**：
   - 学会了分析递归算法的时间复杂度
   - 理解了状态空间大小对搜索算法效率的影响

### 最短步数规律

通过实验发现棋子移动游戏的最短步数规律：

- n=4: 5步
- n=5: 6步
- 推测：对于n≥4，最短步数为n+1步

### GrayCode.java（格雷码问题）

```java
package com.rongx;

import java.util.ArrayList;
import java.util.List;
import java.util.Scanner;

/**
 * 实验二：格雷码问题
 * 使用递归算法生成格雷码
 */
public class GrayCode {

    /**
     * 递归生成n位格雷码
     *
     * 算法思想：
     * n位格雷码可以由n-1位格雷码构造：
     * 1. 获取n-1位格雷码序列
     * 2. 在n-1位格雷码每个元素前加"0"（前半部分）
     * 3. 将n-1位格雷码逆序，每个元素前加"1"（后半部分）
     * 4. 拼接两部分
     *
     * @param n 格雷码位数
     * @return 格雷码列表
     */
    public static List<String> generateGrayCode(int n) {
        List<String> result = new ArrayList<>();

        // 递归出口：n=1时直接返回基础格雷码
        if (n == 1) {
            result.add("0");
            result.add("1");
            return result;
        }

        // 递归调用：获取n-1位格雷码
        List<String> prevGrayCode = generateGrayCode(n - 1);

        // 构造前半部分：在每个格雷码前加"0"
        for (String code : prevGrayCode) {
            result.add("0" + code);
        }

        // 构造后半部分：逆序遍历，在每个格雷码前加"1"
        for (int i = prevGrayCode.size() - 1; i >= 0; i--) {
            result.add("1" + prevGrayCode.get(i));
        }

        return result;
    }

    /**
     * 格式化输出格雷码
     * 每个位之间用空格隔开
     */
    public static void printGrayCode(List<String> grayCode) {
        for (String code : grayCode) {
            StringBuilder sb = new StringBuilder();
            for (int i = 0; i < code.length(); i++) {
                if (i > 0) {
                    sb.append(" ");
                }
                sb.append(code.charAt(i));
            }
            System.out.println(sb.toString());
        }
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        int m = scanner.nextInt();  // 测试例个数

        for (int t = 0; t < m; t++) {
            int n = scanner.nextInt();

            // 生成格雷码
            List<String> grayCode = generateGrayCode(n);

            // 输出格雷码
            printGrayCode(grayCode);

            // 测试例之间用空行隔开（最后一个测试例后无空行）
            if (t < m - 1) {
                System.out.println();
            }
        }

        scanner.close();
    }
}
```

### ChessMove.java（棋子移动游戏）

```java
package com.rongx;

import java.util.*;

/**
 * 实验二：棋子移动游戏
 * 使用BFS广度优先搜索算法求解最短路径
 */
public class ChessMove {

    private int n;                    // 棋子数量（每种颜色）
    private String initialState;      // 初始状态
    private String targetState;       // 目标状态

    /**
     * 构造函数
     * @param n 每种颜色棋子的数量
     */
    public ChessMove(int n) {
        this.n = n;

        // 初始状态: WWW...WBBB...B__ (n个W, n个B, 2个空位)
        StringBuilder init = new StringBuilder();
        for (int i = 0; i < n; i++) {
            init.append('W');
        }
        for (int i = 0; i < n; i++) {
            init.append('B');
        }
        init.append("__");
        this.initialState = init.toString();

        // 目标状态: __WBWBWB... (2个空位, 然后n组WB)
        StringBuilder target = new StringBuilder();
        target.append("__");
        for (int i = 0; i < n; i++) {
            target.append('W');
            target.append('B');
        }
        this.targetState = target.toString();
    }

    /**
     * 使用BFS求解棋子移动问题的最短路径
     * @return 解路径列表，如果无解返回null
     */
    public List<String> solve() {
        // BFS队列
        Queue<String> queue = new LinkedList<>();
        // 记录已访问状态及其前驱状态
        Map<String, String> parent = new HashMap<>();

        queue.offer(initialState);
        parent.put(initialState, null);

        while (!queue.isEmpty()) {
            String current = queue.poll();

            // 找到目标状态
            if (current.equals(targetState)) {
                return reconstructPath(parent, current);
            }

            // 获取所有可能的下一状态
            List<String> nextStates = getNextStates(current);
            for (String next : nextStates) {
                if (!parent.containsKey(next)) {
                    parent.put(next, current);
                    queue.offer(next);
                }
            }
        }

        return null;  // 无解
    }

    /**
     * 获取当前状态所有可能的下一状态
     */
    private List<String> getNextStates(String state) {
        List<String> nextStates = new ArrayList<>();
        int emptyPos = state.indexOf('_');

        // 尝试移动所有可能的相邻两个棋子
        for (int i = 0; i < state.length() - 1; i++) {
            // 检查位置i和i+1是否都是棋子（不是空位）
            if (state.charAt(i) != '_' && state.charAt(i + 1) != '_') {
                // 检查是否可以移动（必须跳过至少一个棋子）
                int minPos = Math.min(i, emptyPos);
                int maxPos = Math.max(i, emptyPos);

                boolean hasJump = false;
                for (int j = minPos + 1; j < maxPos; j++) {
                    if (state.charAt(j) != '_') {
                        hasJump = true;
                        break;
                    }
                }

                if (hasJump) {
                    String newState = move(state, i, emptyPos);
                    if (newState != null) {
                        nextStates.add(newState);
                    }
                }
            }
        }

        return nextStates;
    }

    /**
     * 执行移动操作
     */
    private String move(String state, int from, int to) {
        char[] chars = state.toCharArray();

        char c1 = chars[from];
        char c2 = chars[from + 1];

        chars[from] = '_';
        chars[from + 1] = '_';

        chars[to] = c1;
        chars[to + 1] = c2;

        return new String(chars);
    }

    /**
     * 从BFS结果重建路径
     */
    private List<String> reconstructPath(Map<String, String> parent, String target) {
        List<String> path = new ArrayList<>();
        String current = target;
        while (current != null) {
            path.add(0, current);
            current = parent.get(current);
        }
        return path;
    }

    /**
     * 格式化输出状态
     */
    public static String formatState(String state) {
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < state.length(); i++) {
            if (i > 0) {
                sb.append(" ");
            }
            sb.append(state.charAt(i));
        }
        return sb.toString();
    }

    /**
     * 显示移动过程
     */
    public static void showMoves(List<String> path) {
        if (path == null || path.isEmpty()) {
            System.out.println("No solution found");
            return;
        }

        System.out.println("Initial: " + formatState(path.get(0)));

        for (int i = 1; i < path.size(); i++) {
            String prev = path.get(i - 1);
            String curr = path.get(i);

            // 找出移动的两个棋子
            int fromPos = -1, toPos = -1;
            for (int j = 0; j < prev.length(); j++) {
                if (prev.charAt(j) != '_' && curr.charAt(j) == '_') {
                    fromPos = j;
                    break;
                }
            }
            for (int j = 0; j < curr.length(); j++) {
                if (curr.charAt(j) != '_' && prev.charAt(j) == '_') {
                    toPos = j;
                    break;
                }
            }

            System.out.println("Step " + i + ": Move [" + fromPos + "," + (fromPos+1) + "] -> [" + toPos + "," + (toPos+1) + "]");
            System.out.println("         " + formatState(curr));
        }

        System.out.println("\nTotal: " + (path.size() - 1) + " steps");
    }
```
