---

## 一、实验目的

1. 加深对分治算法的基本思想、基本步骤和一般形式的理解，掌握分治算法设计的基本方法。
2. 用分治法设计L型组件填图问题的算法，分析其复杂性，并实现。
3. 用分治法设计求数列中的第1～k小元素的算法，分析其复杂性，并实现。

## 二、实验内容

### （一）L型组件填图问题

**问题描述**：设B是一个n×n棋盘，n=2^k (k=1,2,3,...)。用分治法设计一个算法，使得用若干个L型条块可以覆盖住B的除一个特殊方格外的所有方格。其中，一个L型条块可以覆盖3个方格，且任意两个L型条块不能重叠覆盖棋盘。

**输入格式**：
- 第一行：正整数m，表示测试例个数
- 每个测试例包含：
  - 第一行：正整数n，表示棋盘大小（n=2^k）
  - 第二行：两个整数row和col，表示特殊方格的位置（0-indexed）

**输出格式**：对于每个测试例，输出一个被L型条块覆盖的n×n棋盘。每个L型条块用不同的数字标记，特殊方格用"A"表示。

### （二）求数列中的第1～k小元素

**问题描述**：设计算法实现在一个具有n个互不相同元素的数组A[1…n]中找出所有前k个最小元素的问题，这里k不是常量，即它是输入数据的一部分。要求算法的时间复杂性为Θ(n)。

**输入格式**：
- 第一行：正整数m，表示测试例个数
- 每个测试例包含三行：
  - 第一行：正整数n，表示元素的个数
  - 第二行：n个整数，整数之间用一个空格隔开
  - 第三行：正整数k，表示求前k个最小元素

**输出格式**：对于每个测试例输出一行，由k个整数组成，表示前k个最小元素（从小到大排序）。两个测试例的输出数据之间用一个空行隔开，最后一个测试例后无空行。

## 四、实验步骤

### （一）L型组件填图问题

#### 1. 算法设计分析

**分治策略核心思想**：

将原问题分解成若干个相互独立且与原问题性质相同的子问题，用同样的方法解这些子问题并将这些子问题的解组合起来得到原问题的解。

**分治步骤**：
1. **分解**：将2^k × 2^k棋盘分成4个2^(k-1) × 2^(k-1)子棋盘
2. **处理**：用一个L型条块覆盖3个不含特殊方格的子棋盘中心交汇处
3. **递归**：对4个子棋盘分别递归处理

**为什么这样分治？**：
- 每次分割后，特殊方格必在其中一个子棋盘
- 其他3个子棋盘的中心交汇处恰好形成L形
- 用一个L型条块覆盖后，每个子棋盘都有了"特殊方格"
- 问题规模减半，可以递归处理

#### 2. 算法图示

**棋盘分割示意**：

```
┌─────────┬─────────┐
│ 左上    │ 右上    │
│ (0,0)   │ (0,mid) │
├─────────┼─────────┤
│ 左下    │ 右下    │
│(mid,0)  │(mid,mid)│
└─────────┴─────────┘
```

**L型条块覆盖策略**：

```
特殊方格在左上子棋盘：
┌───┬───┐
│ S │ 1 │    S = 特殊方格
├───┼───┤    1 = L型条块编号
│ 1 │ 1 │    L型覆盖右上、左下、右下中心
└───┴───┘

特殊方格在右上子棋盘：
┌───┬───┐
│ 1 │ S │
├───┼───┤
│ 1 │ 1 │    L型覆盖左上、左下、右下中心
└───┴───┘

特殊方格在左下子棋盘：
┌───┬───┐
│ 1 │ 1 │
├───┼───┤
│ S │ 1 │    L型覆盖左上、右上、右下中心
└───┴───┘

特殊方格在右下子棋盘：
┌───┬───┐
│ 1 │ 1 │
├───┼───┤
│ 1 │ S │    L型覆盖左上、右上、左下中心
└───┴───┘
```

#### 3. 核心代码实现

```java
/**
 * 递归填充棋盘
 * 
 * @param topRow 子棋盘左上角行坐标
 * @param topCol 子棋盘左上角列坐标
 * @param size 当前子棋盘大小
 * @param specialRow 特殊方格所在行
 * @param specialCol 特殊方格所在列
 */
private void tileBoard(int topRow, int topCol, int size, 
                       int specialRow, int specialCol) {
    
    // 递归出口：当棋盘大小为1时，只有一个方格，无需填充
    if (size == 1) {
        return;
    }

    // 分解：将当前棋盘分成4个子棋盘
    int half = size / 2;
    int currentTile = tileCount++;

    // 情况1：特殊方格在左上子棋盘
    if (specialRow < topRow + half && specialCol < topCol + half) {
        // 用L型条块覆盖其他3个子棋盘的中心交汇处
        board[topRow + half - 1][topCol + half] = currentTile;
        board[topRow + half][topCol + half - 1] = currentTile;
        board[topRow + half][topCol + half] = currentTile;
        
        // 递归处理4个子棋盘
        tileBoard(topRow, topCol, half, specialRow, specialCol);
        tileBoard(topRow, topCol + half, half, topRow + half - 1, topCol + half);
        tileBoard(topRow + half, topCol, half, topRow + half, topCol + half - 1);
        tileBoard(topRow + half, topCol + half, half, topRow + half, topCol + half);
    }
    // 其他3种情况类似处理...
}
```

#### 4. 算法复杂度分析

**时间复杂度**：
- 递归关系：T(k) = 4T(k-1) + O(1)
- 解得：T(k) = O(4^k) = O(n²)
- 解释：棋盘有n²个方格，每个方格都需要被访问一次

**空间复杂度**：
- O(n²)：棋盘存储
- O(k)：递归栈深度（k = log₂n）

**L型条块数量**：
- 设棋盘大小为n=2^k，需要L型条块数量为：(n²-1)/3 = (4^k-1)/3
- 例如：n=2时需要1个，n=4时需要5个，n=8时需要21个

### （二）求数列中的第1～k小元素

#### 1. 算法设计分析

**问题分析**：

题目要求时间复杂度Θ(n)，需要分析不同方法的复杂度：

| 方法 | 思路 | 时间复杂度 | 是否满足 |
|------|------|------------|----------|
| 排序法 | 排序后取前k个 | O(n log n) | ❌ 超出要求 |
| SELECT k次 | 每次找第i小 | Θ(kn) | ❌ k接近n时变成O(n²) |
| 本方法 | 找第k小后遍历收集 | Θ(n) | ✓ 满足要求 |

**核心思想**：

前1~k小元素 = 所有小于等于第k小元素的元素

因为元素互不相同，所以恰好有k个元素小于等于第k小元素。

**算法流程**：
1. 使用快速选择找到第k小元素（时间复杂度Θ(n)）
2. 遍历数组，收集所有小于等于第k小元素的元素（时间复杂度O(n)）
3. 对结果排序输出（不计入时间复杂度）

#### 2. 快速选择算法原理

**快速选择是快速排序的变种**：
- 快速排序：递归处理两个分区
- 快速选择：只递归处理包含目标元素的一个分区

**算法步骤**：
1. 选择一个基准元素（pivot）
2. 将数组分成两部分：小于pivot和大于pivot
3. 判断第k小元素在哪个分区
4. 只递归处理包含目标元素的分区

**分区示意**：

```
原始数组: [56, 34, 22, 7, 16, 95, 46, 37, 81, 12]
基准: 12 (最右元素)

分区过程:
i=-1, j=0: 56 > 12, 不交换
i=-1, j=1: 34 > 12, 不交换
...
i=-1, j=9: 12 <= 12, 交换arr[0]和arr[9]

分区后: [12, 34, 22, 7, 16, 95, 46, 37, 81, 56]
基准位置: 0
```

#### 3. 核心代码实现

```java
/**
 * 快速选择递归辅助方法
 */
private static int quickSelectHelper(int[] arr, int left, int right, int k) {
    // 递归出口：只有一个元素
    if (left == right) {
        return arr[left];
    }

    // 分区操作
    int pivotIndex = partition(arr, left, right);

    // 判断目标位置
    if (k == pivotIndex) {
        return arr[pivotIndex];  // 找到目标
    } else if (k < pivotIndex) {
        return quickSelectHelper(arr, left, pivotIndex - 1, k);  // 左分区
    } else {
        return quickSelectHelper(arr, pivotIndex + 1, right, k);  // 右分区
    }
}

/**
 * 找出前k个最小元素
 */
public static int[] findKSmallest(int[] arr, int k) {
    // 步骤1：找到第k小元素
    int kthSmallest = quickSelect(arr, k);

    // 步骤2：收集所有小于等于第k小元素的元素
    int[] result = new int[k];
    int index = 0;
    for (int num : arr) {
        if (num <= kthSmallest) {
            result[index++] = num;
        }
    }

    // 步骤3：排序输出（不计入时间复杂度）
    Arrays.sort(result);
    return result;
}
```

#### 4. 改进的SELECT算法（扩展内容）

**问题**：标准快速选择最坏情况是O(n²)

**改进方法**：使用"中位数的中位数"作为基准

**算法步骤**：
1. 将数组分成n/5组，每组5个元素
2. 找出每组的中位数
3. 递归找出这些中位数的中位数
4. 用这个"中位数的中位数"作为基准

**为什么选择5？**：
- 每组5个元素，找中位数最多6次比较
- 保证至少有3n/10个元素小于基准
- 保证至少有7n/10个元素大于基准
- 递归深度最多log(n)，保证O(n)最坏时间复杂度

**时间复杂度**：T(n) = T(n/5) + T(7n/10) + O(n) = O(n)

#### 5. 算法复杂度分析

**时间复杂度**：
- 平均情况：T(n) = T(n/2) + O(n) = O(n)
- 最坏情况：T(n) = T(n-1) + O(n) = O(n²)（标准快速选择）
- 改进算法：T(n) = O(n)（最坏情况）

**空间复杂度**：
- O(n)：存储结果数组
- O(log n)：递归栈深度（平均情况）

### 分治算法特点总结

1. **子问题相互独立**：4个子棋盘互不影响；两个分区独立处理
2. **子问题与原问题性质相同**：都是L型填充问题或选择问题
3. **递归出口明确**：棋盘大小为1或数组只有一个元素
4. **组合方式不同**：L型填充直接组合；选择问题需要遍历收集

### LShapeTiling.java（L型组件填图问题）

```java
package com.rongx.exp3;

import java.util.Scanner;

/**
 * 实验三：L型组件填图问题
 * 使用分治法填充棋盘
 */
public class LShapeTiling {

    private int[][] board;
    private int tileCount = 1;

    public void solve(int size, int specialRow, int specialCol) {
        board = new int[size][size];
        tileCount = 1;
        board[specialRow][specialCol] = 0;
        tileBoard(0, 0, size, specialRow, specialCol);
    }

    private void tileBoard(int topRow, int topCol, int size, 
                           int specialRow, int specialCol) {
        if (size == 1) return;

        int half = size / 2;
        int currentTile = tileCount++;

        // 特殊方格在左上
        if (specialRow < topRow + half && specialCol < topCol + half) {
            board[topRow + half - 1][topCol + half] = currentTile;
            board[topRow + half][topCol + half - 1] = currentTile;
            board[topRow + half][topCol + half] = currentTile;
            
            tileBoard(topRow, topCol, half, specialRow, specialCol);
            tileBoard(topRow, topCol + half, half, topRow + half - 1, topCol + half);
            tileBoard(topRow + half, topCol, half, topRow + half, topCol + half - 1);
            tileBoard(topRow + half, topCol + half, half, topRow + half, topCol + half);
        }
        // 其他情况类似处理...
    }

    public void printBoard() {
        for (int i = 0; i < board.length; i++) {
            StringBuilder sb = new StringBuilder();
            for (int j = 0; j < board.length; j++) {
                if (j > 0) sb.append(" ");
                sb.append(board[i][j] == 0 ? "A" : board[i][j]);
            }
            System.out.println(sb.toString());
        }
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int m = scanner.nextInt();
        for (int t = 0; t < m; t++) {
            int n = scanner.nextInt();
            int row = scanner.nextInt();
            int col = scanner.nextInt();
            LShapeTiling tiling = new LShapeTiling();
            tiling.solve(n, row, col);
            tiling.printBoard();
            if (t < m - 1) System.out.println();
        }
        scanner.close();
    }
}
```

### KSmallestElements.java（第1~k小元素问题）

```java
package com.rongx.exp3;

import java.util.Arrays;
import java.util.Scanner;

/**
 * 实验三：求数列中的第1～k小元素
 * 使用快速选择算法
 */
public class KSmallestElements {

    public static int quickSelect(int[] arr, int k) {
        int[] copy = copyArray(arr);
        return quickSelectHelper(copy, 0, copy.length - 1, k - 1);
    }

    private static int quickSelectHelper(int[] arr, int left, int right, int k) {
        if (left == right) return arr[left];
        
        int pivotIndex = partition(arr, left, right);
        
        if (k == pivotIndex) return arr[pivotIndex];
        else if (k < pivotIndex) return quickSelectHelper(arr, left, pivotIndex - 1, k);
        else return quickSelectHelper(arr, pivotIndex + 1, right, k);
    }

    private static int partition(int[] arr, int left, int right) {
        int pivot = arr[right];
        int i = left - 1;
        for (int j = left; j < right; j++) {
            if (arr[j] <= pivot) {
                i++;
                swap(arr, i, j);
            }
        }
        swap(arr, i + 1, right);
        return i + 1;
    }

    public static int[] findKSmallest(int[] arr, int k) {
        int kthSmallest = quickSelect(arr, k);
        int[] result = new int[k];
        int index = 0;
        for (int num : arr) {
            if (num <= kthSmallest) result[index++] = num;
        }
        Arrays.sort(result);
        return result;
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int m = scanner.nextInt();
        for (int t = 0; t < m; t++) {
            int n = scanner.nextInt();
            int[] arr = new int[n];
            for (int i = 0; i < n; i++) arr[i] = scanner.nextInt();
            int k = scanner.nextInt();
            int[] result = findKSmallest(arr, k);
            printArray(result);
            if (t < m - 1) System.out.println();
        }
        scanner.close();
    }
}
```
